# Väestöennuste

Koodilla voi tuottaa kuntien ja kaupungien väestöennusteita kunta ja aluetasolla. Koodi pohjautuu https://github.com/Porvoon-kehitysyksikko/Vaestoennuste tiedostoihin, joita on muokattu vastaamaan käyttötarpeita.
Python-koodia voi ajaa https://jupyter.org/ ohjelmistolla. Lisenssi on MIT-lisessi. Koodi pohjautuu aluperin Seppo Laakson SAS-koodiin, josta vuonna 2021 tehtiin python-versio. V
ersio tuli päivittää, sillä koodi ja python-kirjastot vaativat päivitystä.

1_VKENN.ipynb = Vanhan asuntokannan ennusteen ajaminen aluetasolla (esim. Tilastoeskuksen pienalueet) <br>
2_UTENN.ipynb = Uustuotannon ennusteen ajaminen (Tulee määritellä kaavoitetut alueet ja kem per alue) <br>
3_SEUDUNENN.ipynb = Kuntatason ennuste <br>
4_ENNtasmays.ipynb = Jos halutaan täsmäyttää aluetaso vastaamaan kuntatason kokonaisennustetta. <br>
