# Väestöennuste

Koodilla voi tuottaa kuntien ja kaupungien väestöennusteita kunta ja aluetasolla. Koodi pohjautuu https://github.com/Porvoon-kehitysyksikko/Vaestoennuste tiedostoihin, joita on muokattu vastaamaan käyttötarpeita.
Python-koodia voi ajaa https://jupyter.org/ ohjelmistolla. Lisenssi on edelleeen MIT-lisessi. Koodi pohjautuu aluperin Seppo Laakson SAS-koodiin, josta vuonna 2021 tehtiin python-versio. Versio tuli päivittää, sillä koodi ja python-kirjastot vaativat päivitystä.
